
Quiz 🚀

A fun and interactive Python console-based quiz game that tests your general knowledge!
Answer 10 multiple-choice questions and see your final score with personalized feedback.
Perfect for beginners learning Python and anyone who enjoys quick quizzes.

Features:

- 10 general knowledge questions covering science, geography, history, and nature.
- Type answers using a, b, c, or d.
- Immediate feedback after each question: ✅ Correct / ❌ Wrong.
- Final score with personalized comments:
  - Average, Good, Very Good, or Perfect Score 🎯
- Simple console interface — no extra libraries required.

How to Run:

1. Make sure Python is installed on your computer.
2. Clone or download this repository:

   git clone https://github.com/SnowBitLabs/QuizProject.git

3. Open a terminal or command prompt in the project folder.
4. Run the quiz:

   python Quiz.py

5. Follow the on-screen instructions to answer questions.
